About this Project

Ever do the [100daysofcode.com] challenge and find yourself skipping, forgetting which day, etc?

Well this project aims to correct that with friendly reminders, a simple way to keep track of what day you are on, and will allow you to tweet directly from it (hopefully!).

This is an Opensource project designed by a group of aspiring software engineers learning about collaboration on Github. We hope you find it a really useful tool to track your progress through 100daysofcode.
